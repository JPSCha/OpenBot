# How to run
1°
``
git clone https://github.com/JPSCha/OpenBot
``

2°
``
yarn add i
``

3° configure .env

``
TOKEN=YOUR_TOKEN
CLIENT_ID=CLIENT_ID
PREFIX=PREFIX
``

4°

- ☀️ yarn build - To build the project in Typescript, the compiled files will be in the /build folder
- 🌑 yarn start - to start the project in build/app.js
